# Quest Desk — Техническая документация

Добро пожаловать в документацию информационной системы для организации настольных ролевых игр.

Quest Desk помогает игрокам и мастерам вести персонажей, управлять сюжетом, планировать сессии и систематизировать игровой процесс.

## Ключевые возможности

- Цифровой лист персонажа с автоматическим расчётом модификаторов (D&D 5e).
- Управление квестами и NPC для мастера.
- Планировщик сессий с уведомлениями.
- Общий журнал приключений и личные заметки.
- Разграничение прав доступа.

## Структура документации

- **О проекте** — бизнес-цели, стейкхолдеры, риски.
- **Требования** — функциональные и нефункциональные.
- **Архитектура** — модели данных, выбор хранилища, асинхронное взаимодействие.
- **API** — описание эндпоинтов.
- **Проектные артефакты** — RACI, интервью, BPMN.
- **Релизы** — дорожная карта MVP и следующих версий.
