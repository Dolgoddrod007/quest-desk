# Версия 2.0 (универсальная платформа)

**Цель:** Сделать систему универсальной для разных игровых систем и добавить инструменты для глубокого погружения.

## Ключевые функции

- **Конструктор игровых систем:**
  - Возможность создавать собственные шаблоны персонажей под любую НРИ.
  - Настройка полей, атрибутов, формул расчёта.
  - Поддержка множества систем (D&D, Pathfinder и др.).
- **Расширенные инструменты мастера:**
  - Карта мира (интерактивная, с отметками локаций).
  - Таймлайн событий кампании.
  - Генератор случайных имён и характеристик (для быстрого создания NPC).
  - Подготовка боевых сцен (инициатива, отслеживание хитов врагов).
- **Дайс-роллер с историей бросков:**
  - Поддержка любых комбинаций (2d6+3, d20+5 и т.д.).
  - Сохранение истории с привязкой к сессиям.
  - Автоматический учёт модификаторов из листа персонажа.
- **Социальные функции:**
  - Поиск игроков и мастеров (доска объявлений).
  - Публичные кампании (возможность поделиться своим миром).
  - Система отзывов и рейтингов.
- **Мобильное приложение:**
  - Нативные приложения для iOS и Android.
  - Офлайн-доступ к листам персонажей.

## Технические улучшения

- Внедрение WebSocket для синхронизации в режиме реального времени.
- Расширенная система прав доступа (гостевые роли, временный доступ).
