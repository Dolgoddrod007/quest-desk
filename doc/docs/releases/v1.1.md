# Версия 1.1 (доработка интерфейса и новые функции)

**Цель:** Доработка интерфейса и добавление востребованных функций на основе обратной связи от первых пользователей.

## Новые функции

- **Шаблоны NPC:** возможность сохранять часто используемых NPC как шаблоны для быстрого создания.
- **Быстрый поиск:** поиск по квестам, NPC и заметкам.
- **Расширенный лист персонажа:**
  - Добавление заклинаний (список с ячейками).
  - Учёт веса инвентаря (автоматический подсчёт).
  - Дополнительные поля (раса, класс, уровень, опыт).
- **Импорт/экспорт данных:**
  - Экспорт листа персонажа в PDF (для печати).
  - Импорт персонажей из JSON (из других приложений).
- **Уведомления:**
  - Напоминания о предстоящих сессиях (на email или в интерфейсе).
  - Оповещения о новых записях в журнале.
- **Интеграция с внешними API:**
  - Загрузка данных о монстрах из SRD (для быстрого создания NPC).
  - Подсказки по заклинаниям из открытых источников.

## Технические улучшения

- Оптимизация производительности.
- Улучшение адаптивности для мобильных устройств.
- Внедрение кеширования для ускорения загрузки.
