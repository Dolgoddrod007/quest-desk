# Минимально жизнеспособный продукт (MVP)

**Цель:** Предоставить базовый функционал для одной игровой группы в веб-приложении.

## Основные функции

- Регистрация, аутентификация, управление профилем.
- Создание партии и управление участниками (приглашения по ссылке).
- Лист персонажа (основные характеристики, навыки, инвентарь) с возможностью редактирования.
- Автоматический расчёт модификаторов (на базе D&D 5e) с ручной правкой.
- Мастерский раздел: создание и редактирование квестов и неигровых персонажей (NPC).
- Планировщик сессий (добавление событий, отметка участия).
- Общий журнал сессий (текстовые записи с возможностью комментирования).
- Разграничение прав доступа (игрок видит только своего персонажа и общую информацию, мастер — всё).
- Логирование ключевых событий (регистрация, вход, создание/удаление партий, персонажей, квестов).

## Не реализуются в этой версии

- Шаблоны NPC (быстрое сохранение).
- Быстрый поиск по материалам.
- Расширенный лист персонажа (заклинания, вес инвентаря, дополнительные поля).
- Импорт/экспорт данных (PDF, JSON).
- Уведомления и напоминания.
- Интеграция с внешними API (SRD).
- Конструктор игровых систем, карта мира, дайс-роллер.
- Мобильное приложение.
