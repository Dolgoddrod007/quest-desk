## Обоснование использования DMN

При создании персонажа вызывается таблица решений «Определить стартовые параметры по классу». Она возвращает кость хитов и базовое HP по выбранному классу.

**Почему DMN:**
- Правила D&D 5e фиксированы (Варвар → d12, Волшебник → d6) — идеально для таблицы соответствий.
- В версии 2.0 планируется поддержка нескольких игровых систем; DMN позволяет менять данные без изменения кода.
- Партнёры (издатели) могут верифицировать таблицы в Camunda Modeler, не заглядывая в код.
