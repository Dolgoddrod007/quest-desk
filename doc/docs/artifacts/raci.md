# Матрица RACI для модуля «Лист персонажа»

| Действие | Владелец продукта | Системный аналитик | Backend-разработчик | Frontend-разработчик | UI/UX-дизайнер |
|----------|-------------------|--------------------|----------------------|-----------------------|----------------|
| Определение правил расчёта | A | R | I | I | I |
| Дизайн интерфейса | A | C | I | C | R |
| Реализация бэкенда | A | C | R | I | I |
| Реализация фронтенда | A | I | C | R | C |
| Тестирование | A | C | C | C | C |
| Утверждение релиза и выпуск | A | I | R | R | I |
| Анонс релиза | A | – | – | – | – |

**Обозначения:**  
- **R** – Responsible (исполнитель)  
- **A** – Accountable (ответственный)  
- **C** – Consulted (консультант)  
- **I** – Informed (информируемый)
