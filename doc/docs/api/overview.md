# API Quest Desk

Полная интерактивная документация доступна через Redoc по маршруту `/api/`.  
Ниже приведены основные конечные точки.

## Страница `/` (главная)

| Элементы | Endpoint | Метод | Параметры | Возвращает |
|----------|----------|-------|-----------|------------|
| Список партий пользователя | `/api/campaigns/` | GET | нет | Массив кампаний, где пользователь – участник или мастер |
| Ближайшие сессии | `/api/sessions/` | GET | `campaign_id` (UUID кампании) | Массив сессий со статусом |
| Текущий пользователь | `/api/users/me` | GET | нет | Данные о пользователе |
| Создать партию (модальное окно) | `/api/campaigns/` | POST | `name`, `description`, `master_id` | Объект созданной кампании |
| Выход из аккаунта | на клиенте | – | – | Удаляет токен из локального хранилища, редирект на `/` |

**Примечание:** Копирование кода приглашения выполняется полностью на клиенте.

## Страница `/parties`

| Элементы | Endpoint | Метод | Параметры | Возвращает |
|----------|----------|-------|-----------|------------|
| Список всех компаний | `/api/campaigns/all/` | GET | `?page=` | Массив кампаний |
| Создать партию (модальное окно) | `/api/campaigns/` | POST | `name`, `description`, `master_id` | Объект созданной кампании |
| Текущий пользователь | `/api/users/me` | GET | нет | Данные о пользователе |
| Присоединение к кампании по коду | `/api/campaigns/join-by-code/` | POST | `campaign_id`, `invite_code` | `{"detail": "Вы успешно присоединились к кампании"}` |
| Выход из аккаунта | на клиенте | – | – | Удаляет токен, редирект на `/` |

**Особенности:**
- Поле поиска на этой странице фильтрует карточки на клиенте (без запроса на сервер).
- Действие «скопировать код приглашения» выполняется полностью на клиенте.
