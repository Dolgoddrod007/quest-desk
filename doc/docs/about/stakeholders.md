# Стейкхолдеры

| Роль | Интересы |
|------|----------|
| Владелец продукта | Приоритеты, бэклог, принятие решений. |
| Команда разработки | Реализация, техническое качество. |
| Инвесторы | Стратегические решения, ROI. |
| Пользователи (мастера и игроки) | Источник требований, обратная связь. |
| Партнёры (издатели НРИ) | Контент, интеграции. |
| Юристы | Соответствие законодательству о персональных данных. |
| Маркетинг и PR | Продвижение, работа с сообществом. |
| Техническая поддержка | Помощь пользователям, сбор обратной связи. |

## Бизнес-цели (SMART)

1. **Сократить время подготовки мастера** на 30% к 18 февраля 2027 за счёт быстрого поиска и шаблонов NPC.
2. **Повысить вовлечённость**: не менее 70% созданных партий проведут минимум 2 сессии через систему к 18 февраля 2027.
3. **Сократить время заполнения листа персонажа** с 30 до 15 минут за счёт авторасчётов к 18 февраля 2027.

## Ключевые метрики (KPI)

| Метрика | Целевое значение | Способ измерения |
|---------|------------------|------------------|
| Функциональная полнота | 100% к дате защиты | Сверка с чек-листом требований |
| Среднее время подготовки мастера | ≤ 70% от исходного | Хронометраж |
| Среднее время заполнения листа персонажа | ≤ 15 мин | Замер при создании |
| Процент завершённых созданий персонажа | ≥ 80% | Юзабилити-тестирование |
| Удовлетворённость мастеров | ≥ 4.0 из 5 | Анкетирование |
| Удовлетворённость игроков | ≥ 4.0 из 5 | Анкетирование |
