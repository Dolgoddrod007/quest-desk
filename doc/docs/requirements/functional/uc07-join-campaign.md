# Use Case 7: Вступление в партию по пригласительной ссылке

| Поле | Описание |
|------|----------|
| **ID** | UC-07 |
| **Название** | Вступление в партию по пригласительной ссылке |
| **Акторы** | Игрок (зарегистрированный, но не участник данной партии), Система |
| **Предусловия** | Игрок авторизован; имеет действительную пригласительную ссылку в партию; ещё не является участником этой партии |
| **Постусловия (успех)** | Игрок добавлен в партию с ролью «Игрок»; партия появляется в списке партий игрока; игрок может создавать персонажа для этой партии |
| **Постусловия (неудача)** | Игрок не добавлен в партию; пригласительная ссылка может быть признана недействительной |

## Основной поток

1. Игрок переходит по пригласительной ссылке (например, полученной от мастера).
2. Система проверяет действительность ссылки (существует ли такая партия, не истёк ли срок).
3. Система отображает страницу с информацией о партии (название, мастер, описание) и кнопкой «Присоединиться».
4. Игрок нажимает «Присоединиться».
5. Система добавляет пользователя в партию с ролью «Игрок».
6. Система перенаправляет пользователя на страницу партии, где он может создать персонажа.

## Альтернативные потоки

**Ссылка истекла или недействительна**  
Точка входа: Шаг 2  
2.1 Система не находит партию по ссылке или срок действия истёк.  
2.2 Сообщение: «Пригласительная ссылка недействительна или истекла. Свяжитесь с мастером для получения новой ссылки».  
2.3 Use Case завершается.

**Пользователь уже в партии**  
Точка входа: Шаг 2  
2.1 Система обнаруживает, что пользователь уже является участником этой партии.  
2.2 Сообщение: «Вы уже состоите в этой партии» и перенаправляет на страницу партии.  
2.3 Use Case завершается.

## Бизнес-правила

- **BR-07-01:** Пригласительная ссылка действительна в течение 7 дней с момента создания.
- **BR-07-02:** Один пользователь может быть участником неограниченного числа партий.
- **BR-07-03:** При вступлении пользователь получает роль «Игрок»; мастер не может быть добавлен по приглашению (только создание партии).
